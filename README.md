# ICM Diaphragm Analysis

This repository contains Python implementations of the Instantaneous Center Method (ICM) applied to steel deck diaphragms.

## Features
- Arc-spot weld modeling
- Screw fastener backbone curves
- Nonlinear force redistribution
## Diaphragm Types
### Homogeneous Diaphragms (SDI DDM)
- Arc-spot welds only
- Organized by deck thickness:
  - 22 ga
  - 20 ga
  - 18 ga
  - 16 ga

### Mixed Diaphragms (SDII)
- Arc-spot welds (structural)
- Screws (sidelap)
